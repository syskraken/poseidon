KRAKEN PRIME — INSTALLATION GUIDE

Step 0 — Install LDPlayer and Clash of Clans (required before anything else).
1. Download LDPlayer (free Android emulator) from the official site:
   https://www.ldplayer.net — click Download and run the installer.
2. Open LDPlayer, sign in to the Google Play Store with a Google account, and
   install Clash of Clans from the Play Store inside the emulator.
3. Open Clash of Clans once and log in to the account you want to farm on
   (link it via Supercell ID so you don't lose progress).
4. Configure LDPlayer for the bot (it will NOT work without this):
   - Settings -> Display tab: resolution 1600x900, DPI 240
   - Settings -> Others tab: ADB Debugging = "Enable local connection"
   - Save and restart LDPlayer.

Step 1 — Get the files.
Option A (terminal): git clone https://github.com/syskraken/krakenprime.git
then cd krakenprime.
Option B (zip): download the repository as a zip from GitHub and extract it.

Step 2 — Run setup.bat.
This installs everything the bot needs, automatically:
- Python 3.11 (the bot requires exactly Python 3.11 — newer versions like 3.12+
  will NOT work)
- Python packages: customtkinter, opencv-python, numpy, Pillow, pytesseract,
  requests, certifi
- Tesseract OCR (used to read gold amounts on screen). A bundled installer is
  included; keep the DEFAULT install path if the manual installer opens.
- ADB (Android Debug Bridge) — downloaded into the bot folder if missing.
setup.bat needs an internet connection.

Step 3 — Run build.bat.
This compiles the bot into a standalone KrakenPrime.exe.

Step 4 — Start the bot.
Go to the dist folder and run KrakenPrime.exe.
(If you are running from source instead, run.bat starts the GUI directly.)

Step 5 — Configure LDPlayer (REQUIRED, the bot will not work without this):
1. Open LDPlayer settings -> Display tab -> set resolution to 1600x900 (240 DPI).
2. Settings -> Others tab -> set "ADB Debugging" to "Enable local connection".
3. Restart LDPlayer after changing these settings.

Step 6 — First launch.
Open Clash of Clans in LDPlayer, stay on your home village screen, then open
KRAKEN PRIME, fill in the Configuration page, and run the guided setup
(Configuration -> Advanced -> RUN FULL GUIDED SETUP).
